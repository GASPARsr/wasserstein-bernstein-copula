# Wasserstein-Bernstein Copula Spatial Simulation Framework

Python implementation of a stochastic spatial simulation framework for modeling hydraulic conductivity in heterogeneous aquifers using local Bernstein copulas, a tri-objective optimization function, and hybrid genetic optimization.

## Features
- Local nonparametric Bernstein copula simulation
- Variogram consistency objective
- Wasserstein-based distribution preservation
- Hybrid optimization-ready framework

## Installation
```bash
pip install -r requirements.txt
```

## Example
```bash
python examples/run_minimal_example.py
```

## Repository structure
- `src/`: core implementation
- `examples/`: reproducible example
- `data/`: benchmark dataset
